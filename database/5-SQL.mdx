## CREATE TABLE
숫자는 SMALL INT 등 여러개 있는데 그냥 NUMBER 씀

문자는 고정됀 문자랑 가변길이 문자가 있는데 그냥 가변길이 문자를 씀, 1이랑 2가 있는데 2를씀, 무조건임 장점으론 4000바이트까지 저장 가가능한데 5바바이트만 쓰면 바이트ㅏㄴ 쓰는등 똑똑함
