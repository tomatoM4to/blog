# 데이텁베이스란 무엇인가?
데이터베이스를 공부하기전 먼저 데이터에 대한 정의부터 시작하겠습니다.

* 데이터: 관찰의 결과로 나타난 정량적 혹은 정석정인 실제 <R>값</R>
* 정보: 데이터에 의미를 부여한 것
* 지식: 사물이나 현상에 대한 이해

자연스럽게 데이터베이스는 데이터, 즉 값을 저장하는 공간이라고 할 수 있습니다. 사실 데이터나 정보를 정의하는것은 추상적인 개념이기도 하고 데이터베이스는 엄밀하게 무엇이 데이터베이스다! 라고 정의돼지 않았습니다. 값을 저장하기 위한 `SoftWare` 라고 생각하면 될것 같습니다.



## 역할
데이터베이스의 역할은 크게 2가지로 나눌수 있습니다.

1. Read: 검색
2. Write: 변경

데이터베이스는 결국 수십, 수억의 데이터를 빠르고 안전하게 저장하고, 검색하고 변경하는것이 목적입니다.
